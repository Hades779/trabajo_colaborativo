comando explicacion
html 5 HTML5 es un estándar que sirve como referencia del software que conecta con la elaboración de páginas web en sus diferentes versiones, define una estructura básica y un código (denominado HTML) para la definición de contenido de una página web, como texto, imágenes, vídeos, juegos, entre otros…
h1 hasta h6 En el código HTML de un sitio web, los encabezados se definen mediante una etiqueta HTML especial, existiendo 6 jerarquías de estos, de H1 hasta H6. La H significa "encabezado" y el número expresa su importancia, disminuyendo su relevancia del H1 a H6. Las etiquetas H de HTML tiene el siguiente aspecto
link El elemento HTML especifica la relación entre el documento actual y un recurso externo. Los usos posibles de este elemento incluyen la definición de un marco relacional para navegación. Este elemento es más frecuentemente usado para enlazar hojas de estilos.
p Definición. El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos. Sus etiquetas son: < p> y < /p> (la de cierre es opcional).
id El atributo global id define un identificador único (ID) el cual no debe repetirse en todo el documento. Su propósito es identificar el elemento al vincularlo (usando un identificador de fragmento), en scripts u hojas de estilo (con CSS).< /td>
class El atributo de clase HTML se utiliza para especificar uno o varios nombres de clase para un elemento HTML. El nombre de la clase puede ser utilizado por CSS y JavaScript para realizar algunas tareas para elementos HTML. Puede usar esta clase en CSS con una clase específica, escriba un punto (.)
div La etiqueta < div> se emplea para definir un bloque de contenido o sección de la página, para poder aplicarle diferentes estilos e incluso para realizar operaciones sobre ese bloque específico. Hemos empleado la etiqueta para crear un bloque, que hará las veces de índice de contenidos.< /td>
span En HTML, la etiqueta span es un elemento contenedor en línea genérico. Las etiquetas span generalmente envuelven secciones de texto con fines de estilo o para agregar atributos a una sección de texto sin crear una nueva línea de contenido< /td>
br El elemento HTML line break < br> produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significante. No utilices < br> para incrementar el espacio entre líneas de texto; para ello utiliza la propiedad margin de CSS o el elemento < p> .< /td>
hr Definición. El elemento HTML < hr> representa un cambio de tema entre párrafos (por ejemplo, un cambio de escena en una historia, un cambio de tema en una sección). En versiones previas de HTML representaba una línea horizontal.< /td>
table Una tabla HTML se crea con una etiqueta de apertura < table> y una etiqueta de cierre < /table>. Dentro de estas etiquetas, los datos se organizan en filas y columnas usando etiquetas < tr> de fila de tabla de apertura y cierre y etiquetas < td> de datos de tabla de apertura y cierre. Las etiquetas de fila de tabla < tr> se utilizan para crear una fila de datos
